📚 Book Store Website

This is a full-stack book store web application developed using Node.js and MongoDB.

Features

- Search Books
- Add to Cart
- Wishlist
- Checkout System
- Dark Mode

Technologies Used

- HTML, CSS, JavaScript
- Node.js, Express.js
- MongoDB

How to Run

1. npm install
2. node server.js
3. Open http://localhost:5000
