📚 Book Store Website

📌 Project Overview

This is a full-stack Book Store web application developed using Node.js, Express, and MongoDB. The system allows users to browse, search, and purchase books easily.

---

🚀 Features

- 🔍 Search Books
- 🛒 Add to Cart
- ❤️ Wishlist
- 🌙 Dark Mode
- 💳 Checkout (COD & Online)

---

🛠 Technologies Used

- HTML, CSS, JavaScript
- Node.js, Express.js
- MongoDB Atlas

---

⚙️ How to Run

1. Install Node.js
2. Run "npm install"
3. Run "node server.js"
4. Open http://localhost:5000
